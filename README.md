# Thales_Group_Manufacturing-Efficiency

AI-Powered Manufacturing Efficiency Classification
# Project Overview

This project introduces an AI-driven system to classify manufacturing efficiency into High, Medium, and Low using machine sensor data, production metrics, and 6G network performance.

It helps industries:
 Detect efficiency drops in real-time
 Reduce production losses
 Enable intelligent decision-making

# Features
 Real-time efficiency prediction, 
 Interactive Streamlit dashboard,
 Machine Learning models (Random Forest, Logistic Regression),
 Feature importance & explainability,
 Machine-level insights and trends,
Integration-ready for smart factories

# Tech Stack
 Python
 Pandas, NumPy
 Scikit-learn
 Matplotlib
 Streamlit
